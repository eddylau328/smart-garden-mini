It attaches the interrupt event of the rotate event and press event, and it saves the callback function pointer for class that will need to use that. (Noted: The design for this class is not ready for multiple RotaryEncoder Object to assign callback functions.)

Parameters
_pressCallback- the callback function when press action is triggered, nothing will parse to this callback function
_rotateCallback- the callback function when rotate action is triggered, the step count will parse to this callback function
